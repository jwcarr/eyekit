def combine_images(images, output_path, image_width=200, image_height=None, v_padding=5, h_padding=5, e_padding=1, auto_letter=True)

Combine image objects together into one larger image. images should be a list of list of Image structure. For example, [[img1, img2], [img3, img4]] results in a 2x2 grid of images. image_width is the desired mm (SVG, PDF, EPS) or pixel (PNG) width of the combined image. If auto_letter is set to True, each image will be given a letter label.
